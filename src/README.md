# 前言

这是一个可以直接放到 GitHub 使用的 **mdBook** 模板仓库。

你会得到：

- 一套可维护的文档目录结构（`src/` + `SUMMARY.md`）
- 可本地预览的开发体验（`mdbook serve`）
- 可选的 GitHub Pages 自动部署（见 `.github/workflows/deploy.yml`）

## 你应该从哪里开始写

通常推荐：

- 在 `src/guide/` 写“如何使用/如何贡献”等稳定内容
- 在 `src/examples/` 写“可复制的示例”
- 在 `src/appendix/` 放“FAQ/术语表/参考资料”

## 约定（建议）

- 文件名尽量用英文小写 + `-`，标题用中文更友好
- 一章一事：一篇文档解决一个明确问题
- 把“结论”放在前面，把“推导/背景”放在后面
